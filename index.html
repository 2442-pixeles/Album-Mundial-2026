⚙ Configurar sincronización

Seguí estos pasos para sincronizar entre PC y celu:

1. Entrá a jsonbin.io y creá una cuenta gratuita
2. Andá a Account → API Keys y copiá tu Master Key
3. Pegala acá abajo y hacé clic en Crear bin
4. Se genera tu Bin ID — copialo y usalo en el otro dispositivo con Conectar bin existente
